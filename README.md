# 🧭 求职情报站 · AI 职业导航 Agent (v2.0)

一个基于大模型驱动的硬核求职诊断工具。它拒绝废话，通过精准解构目标岗位 JD、深度交叉比对个人简历，为求职者提供透明的加权匹配度计算、核心能力 Gap 盘点，以及周级别的魔鬼突击行动计划。

## 🚀 产品核心痛点与 v2.0 升级

在传统 AI 求职工具中，普遍存在三大痛点：
1. **岗位要求泛泛而谈**：同一个岗位在不同公司要求天差地别，空谈岗位名称没有意义。
2. **匹配得分像黑盒**：AI 随便给出一个“72%”的匹配度，缺乏可信的计算依据。
3. **成长路线沦为正确的废话**：例如“学习SQL”，让求职者看得到却无从下手。

**求职情报站 v2.0 核心解法：**
- **🎯 岗位画像（JD逆向解构）**：用户直接粘贴真实的目标岗位 JD 文本，AI 强行拆解出其最看重的 5 大能力指标及权重，并提炼高频硬考点与典型业务场景。
- **📊 匹配度拆解（算法透明）**：利用标准的数学加权公式展示得分明细表，并重叠绘制岗位与个人的能力雷达图，明明白白查漏补缺。
- **🗺️ 周级魔鬼突击路线**：拒绝画饼！将突击计划精细到 Week，且每一周都强制绑定唯一的**硬性验收标准 (KPI)**（如：完成15道SQL中等题/输出1份包含RAG框架的PRD）。

## ✨ 功能模块

- **Tab 1：🎯 岗位画像** —— 利用 Plotly 动态绘制岗位能力大盘雷达图，可视化高频考点。
- **Tab 2：📊 匹配度分析** —— 透明的加权公式复核，直观的得分明细表格，精准定位核心能力 Gap。
- **Tab 3：🗺️ 周级路线图** —— 带有核心考点、推荐踩坑资源、以及明确交付物/KPI 的行动计划卡片。

## 🛠️ 技术栈

- **前端可视化**：Streamlit
- **数据可视化**：Plotly (交互式动态雷达图)
- **AI 核心驱动**：DeepSeek API (`deepseek-chat`)
- **非结构化解析**：PyPDF2 (支持标准 PDF 简历自动解析提取)
- **核心语言**：Python 3.x
